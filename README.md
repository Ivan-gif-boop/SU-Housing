# Hostel-Rental-Website
This project proposes the design and development of a web-based centralized hostel and accommodation system tailored specifically to the Strathmore University community. The system will provide a live, administrator-managed listing database through which the Office of the Dean of Students can publish, update, and maintain verified hostel listings. Students will be able to search and filter the live listing database by location, price, and amenity; submit inquiries about specific hostels to the administrator; and provide feedback on hostels they have occupied. Administrators will be able to respond to inquiries and act on student feedback by updating or removing listings as required, enabling continuous quality monitoring of approved accommodation facilities. The system will be developed using a PHP and MySQL stack following RESTful API architecture and a three-tier application design.
